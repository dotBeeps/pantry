Key Interactions

  • Session tabs — switch between active sessions to inspect their context windows
  • Stacked bar — visual breakdown of what's eating context. Hover a segment to highlight its section below.
  • Cache panel — live TTL countdown, cached vs uncached split. Green=warm, amber=expiring, red=cold.
  • Expandable sections — click to see individual files and their token costs within each category
  • Click a file in the expanded list → opens it in an editor (separate panel or overlay) for viewing/editing
  • Token counts are approximate — refreshed on each turn or tool call
  • Cache boundary line — in the expanded conversation view, shows where the cache breakpoint sits between turns
  • Data source — session metadata from daemon MCP register_session + harness-reported token counts
