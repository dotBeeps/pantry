Key Interactions

  • File tree — collapsible directories, click file to open in viewer
  • Raw / Preview toggle — raw shows editable markdown, preview renders it
  • Save button — writes file to disk, appears only when modified (orange dot indicator)
  • Active Rules section — shows all .claude/rules/ files that apply to the current context. Click a chip to open that rule file.
  • Breadcrumb path — clickable segments to navigate up the tree
  • Filter input — fuzzy filter the file tree (not full-text search, just filenames)
  • Scope groups — tree organized by component (hoard root, daemon, dragon-forge, global) not raw filesystem
