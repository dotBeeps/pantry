Design Notes

  • Session rail — colored ring around each icon encodes the tier at a glance (green=puppy, blue=dog, amber=ally, ember-orange=dragon)
  • Small dot bottom-right of icon = connection status (green=connected, grey=disconnected)
  • Guard panel shows the selected session's detail — tier selector, tool whitelist/blocklist, and scoped audit log
  • Click a tier button to promote/restrict — confirmation dialog for escalations
  • Tool tags are clickable — toggle individual tools on/off within the tier's defaults
