Interaction Model

  • ✕ on any tag → removes it from the list (with undo toast)
  • "+ add tool..." → autocomplete dropdown of available tools, click to add
  • "+ block tool..." → same autocomplete, adds to blocklist
  • "+ add path glob..." → text input, validates glob syntax on enter
  • Dashed border on tags = user override (not tier default). Easy to spot what you've customized.
  • Tier change resets to tier defaults — overrides are preserved but flagged as potentially stale
  • Path restrictions scope Write/Edit/Bash to specific directories — the coder ally can only touch dragon-forge/
