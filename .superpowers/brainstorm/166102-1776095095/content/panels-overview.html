Psi Panel System — Key Panels

These are the panels that live in the right-side drawer. Click to explore each one in detail.

Q

Quest Board

Live view of dispatched allies — which quests are running, who's working on what, status updates from sending stone check-ins. The kobold-housekeeping task board + quest status combined. Shows ally taxonomy (scout/coder/reviewer/researcher/planner) with colored badges.

G

Dragon Guard

Permission tier display and tool-call audit log. Shows current permission level (Dog/Puppy/Ally/Dragon), recent tool calls with approve/deny status, and lets you promote or restrict sessions. The safety dashboard.

M

Memory Vault

Browse, search, and edit the daemon's memory. Shelves organized by type (observations, decisions, dot-context). Each entry shows timestamp, source session, and content. Private shelf entries flagged with a lock icon. Direct write/delete from here.

X

Context Inspector

View and edit CLAUDE.md, AGENTS.md, rules files, and persona YAML. Tree view of all context files across the hoard. Inline editor with save. See what context each session is carrying — like claude-crusts but integrated into psi.

A

Attention Monitor

Rich attention state beyond the sidebar gauge. Historical graph of attention over time, cost breakdown per tool type, regen rate visualization, rest gate schedule. See when the daemon is spending vs. recovering attention.

L

Session Log

Scrollable history of all daemon sessions — past thought cycles, tool invocations, memory writes. Searchable. Filter by date, event type, body source. The dragon-scroll equivalent: full audit trail of everything the daemon has done.

S

Settings

Daemon configuration editor — persona selection, LLM provider settings, attention pool tuning, body toggles, contract management, psi interface ports. Edit the YAML visually instead of by hand. Restart daemon with new config from here.
